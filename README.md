## 下载

### Docker

```
docker pull apachecn0/quant-learning
docker run -tid -p <port>:80 apachecn0/quant-learning
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install quant-learning
quant-learning <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g quant-learning
quant-learning <port>
# 访问 http://localhost:{port} 查看文档
```
